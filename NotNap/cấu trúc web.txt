===========================
DỰ ÁN WEBSITE G9 TRANG SỨC
===========================

Công nghệ sử dụng:
- Backend: Python FlaskAPI
- Database: Microsoft SQL Server
- Frontend: HTML5, CSS3, JavaScript, Bootstrap 5
- API: Fetch / Axios
- Thông báo: SweetAlert2
- Dashboard: Chart.js
- IDE: Visual Studio Code
- Quản lý source: Git / GitHub
